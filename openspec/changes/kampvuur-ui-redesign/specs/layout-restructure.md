# Spec: Layout Herstructurering

## Beschrijving
Combineer de drie losse kaarten (streak, freezes, weekoverzicht) tot een hero-sectie + week-sectie layout.

## Acceptatiecriteria
- [ ] Hero card combineert: streak getal, langste streak, vlammenpad, en freeze-status
- [ ] Streak getal prominent links/midden met 🔥 icoon
- [ ] Langste streak als compact element in de hero
- [ ] Freeze schilden (🛡️) compact onderaan de hero card
- [ ] Week card blijft apart met navigatie
- [ ] Hero card heeft subtiele amber glow box-shadow
- [ ] App titel gebruikt 🔥 i.p.v. 🏃
