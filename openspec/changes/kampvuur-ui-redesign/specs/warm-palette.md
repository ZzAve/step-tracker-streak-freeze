# Spec: Warm Kleurenpalet & Typografie

## Beschrijving
Vervang het koude navy/indigo kleurenpalet door warme amber/hout tonen. Vervang system fonts door Nunito.

## Acceptatiecriteria
- [ ] Alle CSS custom properties updaten naar warm palet (zie design.md)
- [ ] Google Fonts link voor Nunito toegevoegd
- [ ] Body font-family is Nunito met system-ui fallback
- [ ] Streak getal gradient is wit → amber (niet wit → indigo)
- [ ] Body achtergrond heeft subtiele radial gradient (warme vignette)
- [ ] Kaarten hebben zachte amber glow shadow i.p.v. harde borders
- [ ] Border-radius verhoogd naar 14px/10px
