# Spec: Animaties

## Beschrijving
Voeg micro-interacties en laad-animaties toe voor een levendiger gevoel.

## Acceptatiecriteria
- [ ] Streak getal telt op van 0 naar huidige waarde bij laden (~800ms, eased)
- [ ] Milestone markers verschijnen staggered (100ms delay per marker)
- [ ] Kaarten fade-in met translateY(10px → 0) bij laden
- [ ] Dag-cel hover: zachte amber glow + scale(1.02)
- [ ] Behaalde milestone markers hebben subtiele pulse animatie
